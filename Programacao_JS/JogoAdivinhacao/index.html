Jogo de Adivinhação

Neste exercícios, você irá criar um jogo de adivinhação utilizando JavaScript. O objetivo é desenvolver uma aplicação que permita ao usuário tentar adivinhar um número gerado aleatoriamente pelo programa.Descrição da Atividade Você deve desenvolver uma página web onde o usuário pode tentar adivinhar um número entre 1 e 100. O programa deve fornecer feedback ao usuário sobre suas tentativas, informando se o palpite está muito baixo, muito alto ou correto. Estilize a página para torná-la visualmente atraente.Utilize cores, fontes e layout que melhorem a experiência do usuário. Implemente a lógica do jogo utilizando estruturas condicionais. A lógica deve seguir as seguintes regras:
Gere um número aleatório entre 1 e 100 quando a página for carregada
Quando o usuário enviar um palpite, compare o palpite com o número gerado:
Se o palpite for menor que o número, exiba a mensagem: "Muito baixo" e permita jogar novamente.
Se o palpite for maior que o número, exiba a mensagem: "Muito alto" e permita jogar novamente.
Se o palpite for igual ao número, exiba a mensagem: "Você acertou!" .
