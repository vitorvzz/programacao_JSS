Exercicio 1

Crie um script em que o usuário digitará a nota da prova, a nota do trabalho e a nota da lista de exercícios e apresente se ele foi aprovado, reprovado ou exame. A média para aprovação é 7. Caso a media fique entre 4 e 6.9 ele está de exame. Caso contrário está reprovado. Utilize funções com parâmetros e com retorno. Para o calculo da média ponderada considere: 70% para prova; 20% para trabalho e 10% para lista de exercícios.


Exercicio 2

Crie um script com as seguinte funções: para receber um valor, verifica se é par ou impar.


Exercicio 3

Faça um script que realize a conversão de Graus Celsius para Fahrenheit.


Exercicio 4

Crie uma função que leia um número inteiro N digitado pelo usuário e outra função que calcule e exiba os N primeiros termos da sequência de Fibonacci.


Exercicio 5

Crie um script para gerenciamento simples de um carrinho
- função para adicionar um produto (nome e preço) a um array (lida pelo usuário);
- função que receba o array de produtos e retorne o total do carrinho, o imposto (10% sobre total) e o total com imposto;
- função que exiba a lista de itens, subtotal, imposto e total. Use parâmetros e retornos nas funções conforme adequado.
