DOM

Métodos DOM

Os métodos permitem manipular elementos do documento HTML com JS.

  1. document.getElementByID("nome_do_id") - pega o elemento do HTML via Id.
  2. InnerHTML - Inserir/Alterar conteudo
  3. Mudar estilo - alteração de css via JS.
  4. classList - insere uma class no elemento em questão
    • .add - adiciona uma class no elemento.
    • .remove - remove uma class no elemento.
    • .toggle - alterna entre inserir e remover.
  5. createElement - cira elemento HTML.
  6. remove - remove elemento.
